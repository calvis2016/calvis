Computer Assembly Language Visualizer and Simulator (CALVIS)

CALVIS is a cross-platform application that can simulate, visualize and customize assembly instructions from Intel x86-32 Instruction Set Architecture. CALVIS supports the full complement of x86-32 instruction set. It supports not only basic integer and floating point instructions but advance Single Input/Multiple Data (SIMD) instructions such as MMX, SSE, SSE2 as well. CALVIS shows contents of the assembly components: registers, memory and flags, which were affected during the simulation. CALVIS contains visual illustrations of the instructional process, aiding students in understanding assembly instructions. CALVIS manages customized instructions, and registers, allowing it to be flexible enough to support instructions higher than x86-32.
